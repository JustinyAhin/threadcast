ThreadCast now supports user accounts and public profiles.

## What's new

- **GitHub sign-in** — Sign in with your GitHub account to claim your sessions.
- **Public profiles** — Each user gets a profile page at `/u/username` showing all their shared sessions.
- **Refreshed UI** — Updated typography, spacing, and color palette across the app.

## Under the hood

- Migrated to Cloudflare D1 for persistent storage.
- Added session metadata (tool counts, message counts, timestamps).
