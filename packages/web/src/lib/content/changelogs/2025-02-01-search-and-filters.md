Finding the right session just got a lot easier.

## New features

- **Full-text search** — Search across session titles, messages, and code blocks.
- **Date filters** — Filter sessions by date range to find recent or older conversations.
- **Tool filters** — Filter by which tools were used (Read, Write, Bash, etc.).

## Improvements

- Faster page loads with optimized data fetching.
- Better mobile navigation with a collapsible sidebar.
